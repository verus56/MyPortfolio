## ğŸš€ Getting Started  

### Prerequisites  

- Python 3.x  
- OpenCV (cv2)  
- NumPy  

### Installation  

1. Clone the repository:  

```bash  
git clone https://github.com/verus56/Cheating-Detection-System.git  
cd yolo-cheating-detection-system  
```  

2. Install required dependencies:  

```bash  
pip install opencv-python numpy  
```  

3. Download the YOLO v8 model configuration (`model.cfg`) and pre-trained weights (`model.weights`) from the official YOLO website or trusted sources.  

4. Create a `labels.names` file with class labels corresponding to the model.  

5. Place the `model.cfg`, `model.weights`, and `labels.names` files in the project directory.  

### Running the App  

```bash  
python object_detection.py  
```  

The application will process the video and display the detected objects with bounding boxes and confidence scores.
